## üß© Included Configs

This repository's `config/` directory contains ready-to-use configuration folders for various applications. You can find them in `config/` and copy or symlink the ones you need into `~/.config/`.

Included config directories:
- `btop` ‚Äî Configuration for btop system monitor
- `cava` ‚Äî Cava audio visualizer configuration
- `fish` ‚Äî Fish shell configuration (startup, functions, prompts)
- `foot` ‚Äî Foot terminal emulator configuration
- `fuzzel` ‚Äî Fuzzel launcher configuration
- `hypr` ‚Äî Hyprland compositor configuration (layouts, binds, workspaces)
- `mako` ‚Äî Mako notification daemon configuration
- `nvim` ‚Äî Neovim setup and plugins (uses nightly in my workflow)
- `tmux` ‚Äî Tmux configuration and plugin settings
- `uwsm` ‚Äî UWSM session manager configuration
- `waybar` ‚Äî Waybar status bar configuration (alternative to QuickShell)
- `waypaper` ‚Äî Waypaper wallpaper manager configuration
- `wlogout` ‚Äî Wlogout logout menu configuration
- `zed` ‚Äî Zed editor settings

If you want to apply only a subset of these, copy or symlink the corresponding folders into `~/.config/` instead of copying everything.
